How to?

To use DevHub follow these steps:
  1. Select a task.
  2. Read task description.
  3. Write the code.
  4. To get the input data use "input" variable. It is created automatically.
  5. Make sure at the end of your code the result is in the variable "output".
  6. Put the code in the result field.
  7. Press "Send".
  8. Enjoy!
The example below is the solution for the "A + B" task: 
var a = input[0];
var b = input[1];
var output = a + b;
